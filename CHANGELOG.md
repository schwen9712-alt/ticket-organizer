## 2026-05-23 — 补充中文城市映射（费城/尼斯等），修复中文OTA格式识别失败

**改了什么**
往中文城市→IATA 映射表 CN_CITY_IATA 补充了约 50 个常用城市，修复部分中文 OTA 格式订单无法识别的问题。

**为什么改**
用户反馈这段无法识别：
```
1. AA746   06月04日  费城 - 尼斯蔚蓝海岸  18:15  08:10+1
含税：6334元/人 , 免费托运行李1件
乘机人： 1.DENG/YIRUI  护照号码CN/EG7029264/CN/15OCT00/F/01JUL29
```

诊断结果：
- ✓ 航班行 regex 能匹配（AA746、06月04日、城市、时间都提取出来）
- ✓ 含税价格 regex 能匹配（6334元/人）
- ✓ 证件号 regex 能匹配（DENG/YIRUI 女 生15OCT00 到期01JUL29）
- ✗ **唯一问题：'费城' 和 '尼斯蔚蓝海岸' 不在城市映射表里** → cnCityToIATA 返回 null → from/to 为空 → 航段录入条件 `carrier && flight && from && to && date` 失败 → 整单不录入

中文 OTA 解析基础设施（cnCityToIATA / cnDateToIATA）本来就有，只是城市表不够全。
