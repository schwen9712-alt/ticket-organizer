## 2026-05-23 — 识别「基础运价」并加"不可退改"标识

**改了什么**
识别"基础商务舱""基础经济舱""特价"等基础运价，自动标记为不可退改，订单行显示红色「🚫 不可退改」警示徽章。

**为什么改**
用户需求：「基础运价都是没有办法退改的，所以需要增加标识」

用户的订单：
```
表Y
FENG/SHISHI  女  17DEC97
1.  UA888  P   TU09JUN  SFOPEK ...
2.  UA889  P   WE24JUN  PEKSFO ...
02 PFX04PBZ+*  49151 CNY
基础商务舱        ← 这个"基础"标识没识别
```

之前"基础商务舱"前面的"基础"二字被舱位解析的正则挡掉（只认"商务舱"），基础运价的不可退改属性完全丢失。

**修复方案**
1. **舱位识别支持"基础/特价/促销"前缀**："基础商务舱"→ 舱位=商务舱 + basicFare 标记
2. **全文兜底扫描**：即使"基础X舱"不在独立行，或出现"不可退改/不退不改"等字样，也能识别
3. **新增 basicFare / basicFareLabel 字段**
4. **订单行红色警示徽章**「🚫 不可退改」，点击显示详情提醒

**识别的关键词**
- 基础前缀：基础/特价/促销 + 舱位
- 直接描述：不可退改/不可退/不可改/不退不改/无法退改

**端到端测试通过**
- ✓ "基础商务舱" → 商务舱 + 基础运价 + 标签"基础商务舱"
- ✓ "基础经济舱" → 经济舱 + 基础运价
- ✓ "特价经济" → 基础运价
- ✓ "商务舱"/"头等舱"（无基础）→ 不标记（正确）
- ✓ 行内兜底（同行带"基础商务舱"）也识别

**UI 效果**
基础运价订单的订单行显示红色「🚫 不可退改」徽章，鼠标悬停/点击弹出：
"⚠ 基础运价（基础商务舱）：此票为基础/特价运价，通常不可退票、不可改签。出票前请向客户确认！"

**风险或注意事项**
- ⚠ 识别"基础/特价/促销"+舱位，或明确的"不可退改"字样。如果运价限制用其他说法表达，可能识别不到（可手动）
- ⚠ 这只是提醒标识，不影响价格计算或出票流程
- ⚠ "基础经济舱"这种本来就是廉价舱的，也会标（符合预期，basic economy 确实多数不可退改）

**回滚方式**
从 .backups/ 找上一版覆盖。回滚后基础运价不再识别和标识。
