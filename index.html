📐DISCOUNT RULES 折扣规则
多条件折扣规则：可按 代理 + 航司 + 舱位 + 表 Label + 始发地 + 舱位代码 组合设定折扣。
规则越具体（条件越多）优先级越高。例：表B + OZ + 商务舱 + Z + US始发 = 81% 优先于 季凯 = 88%。
空字段表示匹配所有（通配符）。
🎁PRODUCT DISCOUNTS 产品折扣（最高优先级）
解析 PNR 时，如果文本里出现指定标记（如 产1、产2），自动套用对应折扣并标记为"特殊折扣"。
优先级最高：覆盖所有其他折扣规则。匹配规则按列表顺序，先匹配先用。
